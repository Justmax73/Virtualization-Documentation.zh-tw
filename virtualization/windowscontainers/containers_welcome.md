# Windows 容器文件

Windows 容器所提供的作業系統層級虛擬化，可讓多個隔離的應用程式在單一的系統上執行。 與功能一起隨附的容器執行階段有兩種類型，每一種皆具有不同程度的應用程式隔離。 Windows Server 容器透過命名空間和處理序隔離來完成隔離。 Hyper-V 容器將每個容器封裝在輕量虛擬機器中。 此文件集提供快速入門指南、部署指南和管理作業的技術詳細資料。
