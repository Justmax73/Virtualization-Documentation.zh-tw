# Windows 容器文件

Windows 容器可提供作業系統層級的虛擬化，讓多個隔離的應用程式在單一系統上執行。 此功能包括兩種不同類型的容器執行階段，能提供不同程度的應用程式隔離。 Windows Server 容器會針對命名空間和處理序來實施隔離， 而 Hyper-V 容器則是將各容器封裝於輕量的虛擬機器中。 此文件集提供快速入門指南、部署指南和管理作業的技術詳細資料。
